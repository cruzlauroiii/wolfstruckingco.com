Agentic AI labor platform — walkthrough

One seven-minute run through the Case Study's five required steps, using real infrastructure: Cloudflare Workers + R2 for data and permissions, Claude Opus 4.7 via the Anthropic Messages API for the agentic steps, real OAuth single sign-on with role inference from the live backend. No seeded fixtures — every record created in-frame, narrated by the Aria voice via edge-tts.

1

Worker onboarding

Jordan Vega authenticates via SSO, the Login page reads the callback email and routes by role. The Career Agent form collects role, experience, location, credentials, and goals — every field persisted to R2.

2

Career agent activation

On form submit, the agent fires a live Claude Opus 4.7 call to synthesize a grounded summary — strengths, gaps, suggested next move. No canned response.

3

Job match surface

Five matched listings ranked by location and profile fit. Each card gets a fresh per-job Claude rationale explaining the match.

4

Career agent chat

The worker asks about a specific role. Multi-turn conversation grounded in profile + job, persisted to R2 so the next session continues where this one left off.

5

Employer view (bonus)

A client signs in, opens the Employer view for their open role, and sees every qualifying worker pre-ranked with an AI-written pitch — fresh Claude call per candidate.
